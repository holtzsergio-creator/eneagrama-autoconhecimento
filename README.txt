# 🌿 Eneagrama · Autoconhecimento

Um guia introdutório ao Eneagrama com foco em autoconhecimento e crescimento espiritual, seguindo a tradição dos Padres do Deserto e a conexão entre virtudes e vícios capitais.

## ✨ Funcionalidades

- **Teste Rápido**: 9 perguntas intuitivas com escala Likert de 1 a 5 para identificar seu tipo predominante
- **Perfil Personalizado**: Descrição detalhada do seu tipo com virtude, vício, medo básico, desejo e sugestões de crescimento
- **Catálogo Completo**: Explore todos os 9 tipos do Eneagrama com suas características e caminhos espirituais
- **Armazenamento Local**: Seu resultado é salvo no navegador para consulta futura

## 🎯 Abordagem

Este projeto combina:

- A sabedoria milenar do Eneagrama
- A tradição contemplativa dos Padres do Deserto
- A psicologia das virtudes e vícios capitais
- Um design minimalista e contemplativo

## 🛠️ Tecnologias

- HTML5, CSS3 e JavaScript puro
- LocalStorage para persistência de dados
- Design responsivo e acessível

## 🚀 Como usar

1. Faça o teste respondendo as 9 perguntas
2. Descubra seu tipo principal (ou empate)
3. Explore seu perfil com orientações de crescimento
4. Navegue pelo catálogo para conhecer todos os tipos
